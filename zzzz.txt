Admin introduce API pentru fetch articles salvate in MongoDB prin jobs
Admin poate face CRUD pentru category.
Se introduc articole automat la un interval de timp
User poate subscribe sau unsubscribe la o Categorie. 
In homepage si feed page vedea articole de unde sa abonat si sugentii din articole unde nu sa abonat
Primeste notificari in timp real daca are noutati la articole noi din aceele category unde sa abonat
Poate cauta articole.


Redis, ES, Mysql/MongoDB, Pusher, Horizon, Git, Docker,



Controllers
Api:
 NotificationController.php - Controller API responsabil pentru gestionarea notificărilor utilizatorilor și trimiterea lor către frontend în timp real.
Jobs:
 CalculateArticleScore.php – Job care calculează scorul de relevanță al unui articol pentru a ajuta la ordonarea și prioritizarea feed-ului de știri.
 FetchNewsJob.php – Job care extrage articole din surse externe precum RSS sau API-uri și le trimite spre procesare.
 ProcessArticleJob.php – Job care normalizează datele articolului, verifică duplicatele, detectează categoria și salvează articolul în baza de date.
 ProcessArticlesBatchJob.php – Job care procesează un batch de articole simultan pentru performanță mai bună și le inserează în baza de date.
Observers
 UserObserver.php = Observer care monitorizează evenimentele modelului User și execută automat acțiuni la creare, actualizare sau ștergere.
Events 
 ArticleCreated.php = Event declanșat când un articol nou este creat și folosit pentru broadcasting în timp real către utilizatori pe baza categoriei.

use App\Models\Article;
use App\Events\ArticleCreated;
use Illuminate\Support\Facades\Event;

// Loop and create articles, then trigger event
collect(range(1,5))->each(function($i) {
    $article = Article::create([
        'title' => "Test Article #{$i}",
        'description' => "This is a test notification {$i}",
        'category_id' => 1,      // Make sure this category exists
        'source' => 'Tinker',
        'url' => "https://example.com/test-article-{$i}",
        'published_at' => now(),
    ]);

    // 2️⃣ Trigger the event
    event(new ArticleCreated($article));

    echo "Event triggered for article: {$article->title}\n";
});
